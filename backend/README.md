## Dependencies

* Ruby '2.6.6'
* Rails '~>6.0.3'
* Database: sqlite3

## Set up

* Clone the repo.
```
    git clone https://github.com/HitRails/ExpenseTracker.git
```

* Install dependencies
```
  cd ExpenseTracker
  bundle install
```

* Create database.

```
    rails db:create
```
* Migrate database.

```
    rails db:migrate
```
